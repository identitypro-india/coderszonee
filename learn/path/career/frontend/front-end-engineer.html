Front-End Development

Front-End Development: Build Modern, Responsive Web Applications

  • overview
  • curriculum
  • reviews

Course Descriptions

This curriculum is designed to guide beginners through the entire process of becoming a proficient front-end engineer, from basic HTML and CSS all the way to advanced React, TypeScript, and deployment practices. The hands-on projects provide learners with practical experience to showcase their skills.

To start this Career Path,upgrade your plan.

About this Skill Path

Front-end engineers work closely with designers to make websites beautiful, functional, and fast. This Career Path will teach you the technologies you need to do just that.

    Skills you'll gain

  • Web Development Fundamentals
  • Responsive Design
  • JavaScript Frameworks & APIs
  • Version Control & Deployment
  • Problem-Solving and Debugging
  • React.js, APIs and AJAX
  • Collaboration and Communication
  • Web Accessibility (a11y)
  • Front-End Testing
  • Deployment
